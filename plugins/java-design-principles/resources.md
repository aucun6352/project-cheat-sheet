## Resources

### Essential Books
- **Clean Code** by Robert C. Martin - The bible of clean code writing
- **Effective Java** (3rd Edition) by Joshua Bloch - The essence of Java best practices
- **Design Patterns: Elements of Reusable Object-Oriented Software** by Gang of Four - Classic on design patterns
- **Refactoring: Improving the Design of Existing Code** by Martin Fowler - Refactoring techniques
- **Head First Design Patterns** by Eric Freeman & Elisabeth Robson - Beginner-friendly pattern learning guide

### Online Resources
- [Java Design Patterns](https://java-design-patterns.com/principles) - Comprehensive guide to design principles and patterns
- [SOLID Principles](https://www.baeldung.com/solid-principles) - Baeldung's explanation of SOLID principles
- [Refactoring Guru](https://refactoring.guru/design-patterns/java) - Design patterns and refactoring techniques

### Practice Materials
- [Design Patterns in Java](https://github.com/iluwatar/java-design-patterns) - Real implementation examples on GitHub
- [Clean Code Examples](https://github.com/JuanCrg90/Clean-Code-Notes) - Practical applications of Clean Code principles

### Further Learning
- **Object-Oriented Design & Patterns** by Cay S. Horstmann - Advanced object-oriented design
- **Growing Object-Oriented Software, Guided by Tests** by Steve Freeman - The harmony of TDD and design
