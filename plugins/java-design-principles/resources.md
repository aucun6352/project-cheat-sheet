## Resources

### 필수 서적
- **Clean Code** by Robert C. Martin - 클린 코드 작성의 바이블
- **Effective Java** (3rd Edition) by Joshua Bloch - Java 모범 사례의 정수
- **Design Patterns: Elements of Reusable Object-Oriented Software** by Gang of Four - 디자인 패턴의 고전
- **Refactoring: Improving the Design of Existing Code** by Martin Fowler - 리팩토링 기법
- **Head First Design Patterns** by Eric Freeman & Elisabeth Robson - 초보자 친화적인 패턴 학습서

### 온라인 자료
- [Java Design Patterns](https://java-design-patterns.com/principles) - 설계 원칙과 패턴의 종합 가이드
- [SOLID Principles](https://www.baeldung.com/solid-principles) - Baeldung의 SOLID 원칙 설명
- [Refactoring Guru](https://refactoring.guru/design-patterns/java) - 디자인 패턴과 리팩토링 기법

### 실습 자료
- [Design Patterns in Java](https://github.com/iluwatar/java-design-patterns) - GitHub의 실제 구현 예제
- [Clean Code Examples](https://github.com/JuanCrg90/Clean-Code-Notes) - Clean Code 원칙의 실제 적용 사례

### 추가 학습
- **Object-Oriented Design & Patterns** by Cay S. Horstmann - 객체지향 설계 심화
- **Growing Object-Oriented Software, Guided by Tests** by Steve Freeman - TDD와 설계의 조화
