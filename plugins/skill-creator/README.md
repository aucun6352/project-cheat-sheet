# Skill Creator Plugin

Claude Code를 위한 대화형 스킬 생성 플러그인입니다.

## 📖 개요

이 플러그인은 Claude Code에서 사용할 수 있는 스킬(Skill)을 **지능형으로** 생성할 수 있도록 도와줍니다.

### ✨ 주요 특징

- 🧠 **대화 내용 분석**: 현재 세션의 대화를 자동으로 분석하여 스킬 정보 추출
- 💡 **지능형 추천**: 분석 결과를 바탕으로 스킬 이름, 설명, 내용을 자동으로 제안
- ✏️ **선택형 입력**: 추천된 내용 중 선택하거나 직접 수정 가능
- 📝 **표준화된 구조**: 일관된 품질의 SKILL.md 자동 생성

## 🚀 사용법

### 스킬 생성하기

```bash
/create-skill
```

커맨드를 실행하면 다음과 같이 진행됩니다:

#### 프로세스

1. **대화 분석 (자동)**: 현재 세션의 대화 내용을 분석하여 주요 주제, 코드 예제, 패턴 추출
   - ⚠️ 대화 내용이 부족하면 자동으로 건너뛰고 직접 입력 모드로 전환

2. **정보 수집 (추천형/직접 입력형)**:
   - 스킬 이름, 설명, 사용 시나리오, 핵심 개념, 패턴/예제, 모범 사례, 주의사항 입력
   - 분석 결과가 있으면 추천 내용 제공, 없으면 직접 입력

3. **스킬 생성 (자동)**: 입력한 내용을 바탕으로 표준 SKILL.md 파일 자동 생성

### 생성된 스킬 구조

```
{프로젝트-루트}/
└── {스킬-이름}/
    └── SKILL.md              # 메인 스킬 문서
```

### SKILL.md 구조

```markdown
---
name: skill-name
description: 스킬 설명
---

# 스킬 제목

## When to Use This Skill
- 사용 시나리오 1
- 사용 시나리오 2

## Core Concepts
핵심 개념 설명

## Patterns
코드 예제 및 패턴

## Best Practices
- 모범 사례 1
- 모범 사례 2

## Common Pitfalls
- 주의사항 1
- 주의사항 2
```

## 💡 예제

### API 디자인 대화 후 스킬 생성

```bash
# Claude와 RESTful API 설계에 대해 대화한 후
/create-skill
```

Claude가 대화를 분석하여 다음을 자동 추천:
- **스킬 이름**: `api-design-principles`, `restful-api-patterns` 등
- **설명**: "RESTful API 설계 원칙과 모범 사례..."
- **시나리오**: "새로운 API 엔드포인트 설계 시", "API 구조 리뷰 시" 등

추천 내용을 선택/수정하여 몇 번의 클릭으로 스킬 생성 완료!

## 🎯 스킬 활용

생성된 스킬은 다음과 같이 활용할 수 있습니다:

```
"API를 설계할 때 모범 사례를 알려줘 @api-design-principles/SKILL.md"
```

## 🔧 커스터마이징

생성된 스킬은 언제든지 수정 가능합니다:
- `{스킬-이름}/SKILL.md` 파일을 열어 편집
- 추가 리소스는 `assets/`, `references/` 디렉토리에 구성 가능
